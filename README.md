# Mundo Mascotas Web

**Mundo Mascotas** es una página web sencilla desarrollada con **HTML5** y **CSS3**.  
El proyecto está centrado en mostrar información sobre mis perros y las mascotas, incluir una pequeña galería de imágenes y presentar una estructura web responsive siguiendo el enfoque **mobile first**.

---

## Descripción del proyecto

La página está diseñada para presentar diferentes secciones relacionadas con las mascotas:

- Una **página principal** con información general.
- Una sección sobre **mis compañeros de vida**.
- Un apartado sobre la **importancia de las mascotas**.
- Una **tabla de estadísticas**.
- Un **formulario de contacto**.
- Una **galería de imágenes** en una página adicional.

El objetivo del proyecto es practicar la creación de una página web estructurada, visualmente ordenada y adaptable a distintos tamaños de pantalla.

---

## Tecnologías utilizadas

Este proyecto utiliza las siguientes tecnologías:

- **HTML5** → para la estructura del contenido.
- **CSS3** → para los estilos, colores y diseño responsive.
- **Flexbox básico** → para organizar algunos elementos, como las imágenes y la galería.
- **Media Queries** → para adaptar la página a pantallas más grandes.

## Diseño responsive

La página está diseñada siguiendo el enfoque **mobile first**:

1. Primero se diseña para **pantallas pequeñas**.
2. Después se adapta a pantallas más grandes mediante **media queries**.

Esto permite que la web se visualice correctamente en:

- móviles
- tablets
- ordenadores
