# Practica DOM

**Mundo Mascotas** es una página web sencilla desarrollada con **HTML5** y **CSS3** y **JAVASCRIPT**

---

## Cómo clonar el proyecto

Para descargar el proyecto en tu ordenador:

```bash
git clone (https://github.com/Elegm92/pagina-dom)
```

Después entra en la carpeta del proyecto:

```bash
cd landin_web
```

---

## Cómo ejecutar el proyecto

1. Descarga o clona el repositorio.
2. Abre la pagina con (https://github.com/Elegm92/pagina-dom).
3. Abre el archivo **index.html** en tu navegador.
4. Navega por las diferentes secciones de la página.
